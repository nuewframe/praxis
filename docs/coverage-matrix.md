# Static-analysis coverage matrix

> Generated by `scripts/gen-coverage-matrix.sh` from each probe's actual
> `--include` list. Do not edit by hand — CI (`--check`) fails if it is stale.

Praxis is **language-agnostic in doctrine**. Its *static enforcement* is
**best-effort per language**, exactly as tabulated below. Structural probes
apply to every language; text probes apply to the languages they scan.

## Reading this table

- ✓ — the probe scans this language's files and has markers for it.
- — — not scanned; a violation in this language passes the probe silently.
- Adding a language = extend the `--include` list and markers in the text
  probes, then re-run `gen-coverage-matrix.sh --write`.
